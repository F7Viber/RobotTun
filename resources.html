Robot Runner: Build Your Game

In this workshop, you'll create your own endless runner game in HTML. Let's go!

Your Challenge

Create an HTML endless runner game where a robot jumps over obstacles.

Build it from scratch with these features:

  • A robot character on screen that jumps when you press the spacebar
  • Obstacles appear and move from right to left
  • The game ends when the robot hits an obstacle
  • Display the score as the game progresses
  • The game gets faster over time

Setup (3 Steps)

  1. 1Create a GitHub account
    Sign Up for GitHub
  2. 2Create a new project
    Go to github.com/new and create a repository called robot-runner
    (Make it public and add a README)
  3. 3Start coding
    Click the green Code button, select Codespaces, and create a new file called index.html

How to Publish Your Game Online

Once your game works, follow these steps to share it with the world:

  1. 1Save your code
    Make sure your game code is saved in Codespaces.
  2. 2Commit your changes
    In Codespaces, click the Source Control icon (left sidebar), write a message like "Add my robot runner game", and click Commit & Push.
  3. 3Enable GitHub Pages
    Go to your repository settings:
    • Click Settings at the top of your repository
    • Click Pages in the left sidebar
    • Under "Build and deployment", select main branch
    • Click Save
  4. 4View your game online
    After a few seconds, your game will be live at:
    https://YOUR-USERNAME.github.io/robot-runner/
    (Replace YOUR-USERNAME with your GitHub username)
