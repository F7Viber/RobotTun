Robot Runner: Build Your Game

In this workshop, you'll create your own endless runner game. No experience needed—just creativity and curiosity!

Quick Start (3 steps)

  1. 1Create a GitHub account
    Sign Up for GitHub (It's free and takes 2 minutes)
  2. 2Create a new project
    Go to github.com/new and create a repository called robot-runner
    (Keep it public and add a README)
  3. 3Start coding
    Click the green Code button and select Codespaces
    Then create a new file called index.html and start building!

Your First Challenge

Create an endless runner game where:

  • A robot character appears on screen and can jump when you press the spacebar
  • Obstacles spawn from the right side and move toward the left
  • If the robot collides with an obstacle, the game ends and shows your score
  • As time passes, the game gets faster
  • Display the current score on screen

Start by building:

  • Draw simple shapes (rectangles) for the robot and obstacles
  • Make the robot jump with the spacebar
  • Move obstacles across the screen
  • Check if the robot and obstacle collide
  • Keep track of the score

Tips for Success

  • Start simple: Get the game loop working first, add complexity later.
  • Test constantly: Save your code and refresh to see changes immediately.
  • Break it down: Work on one thing at a time (jumping, then obstacles, then collisions).
  • Use the browser console: Press F12 to see errors and debug.
  • Ask for help: Stuck? Describe what you're trying to do.

What's Next?

Once your game works, try adding:

  • Better graphics and colors
  • Sound effects
  • Different types of obstacles
  • Power-ups or special items
  • A high score leaderboard
  • Mobile controls (touch/swipe)
