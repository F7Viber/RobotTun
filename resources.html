Vibe Coding Workshop

Build and publish your own game in 3 steps using vibe coding and AI.

1Create Your Game

Open github.com/new, create a repository called robot-runner (public), then open it in Codespaces.

Click the Copilot chat window and paste this prompt:

Create an HTML endless runner game where a robot jumps over obstacles.

The game should have:

  • A robot character that jumps when you press spacebar
  • Obstacles that move from right to left
  • Game ends when robot hits an obstacle
  • Display the score on screen
  • Game gets faster over time

Copilot will generate the full game code. Review it and let Copilot refine it as needed.

2Publish Your Game

Once your game works in Codespaces, publish it online:

  1. Commit and push — Click Source Control (left sidebar), commit your changes, and push to GitHub
  2. Enable GitHub Pages
    • Go to your repository on github.com
    • Click Settings
    • Click Pages in the left sidebar
    • Select main branch and click Save
  3. Share your game — Your game is live at: https://YOUR-USERNAME.github.io/robot-runner/

3Refine Your Game

Now the magic happens! Use Copilot to improve your game. Try asking for:

  • Better graphics, colors, or animations
  • Sound effects when jumping or hitting obstacles
  • Different types of obstacles
  • Power-ups or special items
  • A leaderboard or high score tracker
  • Mobile controls (touch/swipe)
  • New difficulty levels

Each change: Ask Copilot, review the code, test it, commit, push, and your live game updates automatically!
