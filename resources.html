Vibe Coding Workshop

Build and publish your own game in 3 steps using vibe coding and AI.

1Set Up GitHub Pages & Create Your Project

  1. Go to github.com/new and create a repository called robot-runner (make it public)
  2. Click the green Code button and select Codespaces to open the editor
  3. In Codespaces, create a new file called index.html

2Open Your Project & Paste the Prompt

Click the Copilot chat window on the right and paste this prompt:

Create an HTML endless runner game where a robot jumps over obstacles.

Copilot will generate the complete game code for you. Review it and let Copilot refine it as needed until the game works.

3Publish Your Game

  1. Commit and push — Click Source Control (left sidebar), commit your changes, and push to GitHub
  2. Enable GitHub Pages
    • Go to your repository on github.com
    • Click Settings
    • Click Pages in the left sidebar
    • Select main branch and click Save
  3. View your game online — Your game is live at: https://YOUR-USERNAME.github.io/robot-runner/

4Refine Your Game

This is where the magic happens

Now use Copilot to improve your game. Try asking for:

  • Better graphics, colors, or animations
  • Sound effects
  • Different types of obstacles
  • Power-ups or special items
  • A leaderboard or high score tracker
  • Mobile controls
  • New difficulty levels

The cycle: Ask Copilot → Review code → Test it → Commit & push → Your live game updates automatically!
