Vibe Coding Workshop

Build and publish your own game in 3 steps using vibe coding and AI.

1Get Set Up

  1. Create a GitHub account — Go to github.com/signup and sign up (it's free)
  2. Create a new project — Go to github.com/new, enter a name for it, make it public
  3. Open in Codespaces — Click the green Code button and select Codespaces

2Ask Copilot to Build Your Game

Click the Copilot chat window on the right and paste this prompt:

Create an HTML endless runner game where a robot jumps over obstacles.

Copilot will generate the complete game code. Review it and let Copilot refine it as needed until it works perfectly.

3Ask Copilot to Publish Your Game

Once your game is working, ask Copilot in the chat:

"Publish this game to GitHub Pages and ask for the link to play your game"

Copilot will handle everything and give you the live link to share!

4Refine Your Game

This is where the magic happens

Now ask Copilot for improvements. Try:

  • "Add better graphics and colors"
  • "Add sound effects"
  • "Add power-ups"
  • "Add a high score tracker"
  • "Make it work on mobile"

Each time: Ask → Review → Test → Ask Copilot to publish → Your live game updates instantly!
