Robot Runner: Student Guide

This project is a simple browser-based game that helps teens explore “vibe coding” through quick visual feedback, simple changes, and creative experimentation.

Why this works well for teens

  • It is fast to open and play.
  • Students can change one thing and immediately see the result.
  • It mixes creativity, logic, and experimentation.
  • It stays low-pressure because the project is small and approachable.

Quick setup guide

  1. Open the project folder in VS Code.
  2. Open the terminal and run: python3 -m http.server 8000
  3. Open your browser at http://127.0.0.1:8000/.
  4. To edit the game, open index.html and change the code.
  5. Refresh the page to preview your changes.

What to try first

  • Change the robot speed.
  • Make the jumps higher or lower.
  • Add a new obstacle style.
  • Change the background colors.
  • Add a score or timer.

Good beginner ideas

Simple Visual Creative Fast feedback

Start with one small change at a time. That helps students learn by building confidence rather than feeling overwhelmed.

Helpful tip

When students are stuck, ask them to change one value and predict what will happen before they refresh the page. That turns coding into a game of experimentation.
